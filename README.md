**AI & Machine Learning Bootcamp**

 **About This Repository**

This repository documents my learning journey through a **3-month AI Bootcamp** offered by a software company, focused on **Artificial Intelligence** and **Machine Learning**. Each week's work — including lab notebooks, exercises, and mini-projects — is organized here to track progress from the fundamentals to more advanced data science tools.

The goal of this bootcamp is to build a strong foundation in Python programming, data handling, and the core libraries used across the AI/ML ecosystem, before moving into machine learning models and real-world projects.

**Week 1 — Python Foundations**

**Virtual Environment Setup**
- Learned what a virtual environment is and why it's essential for managing project dependencies
- Practiced creating, activating, and managing virtual environments
- Wrote and executed sample scripts inside the environment to confirm proper setup

**Sets & Dictionaries**
- Studied Python's `set` and `dictionary` data structures in detail
- Practiced key-value operations, set operations (union, intersection, difference), and their real-world use cases

**File Handling in Python**
- Learned how to read and write files using Python's built-in file handling methods
- Practiced different file modes and safe file-handling techniques

---

## Week 2 — Data Analysis Toolkit

**Pandas & Data Manipulation**
- Explored `Pandas` in depth for loading, cleaning, and analyzing structured data
- Practiced handling missing values, `groupby` aggregations, and statistical summaries
- Worked with a real-world dataset (hourly weather data) to apply these concepts practically

**NumPy**
- Learned array creation, reshaping, and vectorized operations
- Practiced matrix manipulation, normalization, and boolean masking on numerical data

**Matplotlib**
- Learned how to visualize data using Matplotlib
- Practiced plotting trends and summaries derived from Pandas/NumPy analysis
